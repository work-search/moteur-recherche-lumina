1: Comment installer une extension dans Firefox

  1. Ouvrez Firefox et allez dans la page de l'extension de work:.
  2. Cliquez sur "Ajouter à Firefox" à côté de l'extension.
  3. Dans la boîte de dialogue, cliquez sur "Ajouter".
  4. Cliquez sur "J'ai compris" dans la fenêtre qui apparaît (si elle appraraît).

2: Définir work search comme moteur de recherche principal

  1. Cliquez sur le bouton de menu (≡) en haut à droite.
  2. Sélectionnez "Paramètres".
  3. Dans la barre latérale, cliquez sur "Recherche".
  4. Sous "Moteur de recherche par défaut", sélectionnez work search.
